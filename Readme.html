Installing packages:

Configure opkg sources

vi /etc/opkg/base-feeds.conf - This will create a new file that edited in Vi. Press ‘i’ to edit and then paste in the following

 

src/gz all http://repo.opkg.net/rdison/repo/all

src/gz edison http://repo.opkg.net/edison/repo/edison

src/gz core2-32 http://repo.opkg.net/edison/repo/core2-32

 

Press Esc button, write :wq and press enter to save the file.

Setting up Candy machine second + time.

 

A pre-requisite is to have a network that can be shared with your computer and Edison Board.

You can set you laptop up as a disconnected hotspot according to the instructions here.
netsh wlan set hostednetwork mode=allow ssid=Hotspot key=<password> 

netsh wlan start hostednetwork

 

1.      Connect board to Electricity

2.      Connect your computer to the serial usb port on the Edison board (the small one by the border)

3.      Open Device Manager

4.      Find the port number of “Usb Serial Port”

5.      Open Putty

·        Click on Serial checkbox

·        In  Serial Line enter the port that is open (COM5)

·        Speed : 115200

·        Click Open

·        In the window that opens click on enter

·        Login should appear

·        Login with your account and password

 

6.      Setup Wifi

·        configure_edison –wifi

·        select the relevant network

·        At the end of the process you will get a URL with the IP of the Edison board enter it to your browser

7.      Once wifi is connected you can connect with ssh instead of serial

8.      Connect USB Hub to Edison “Large” USB Port

9.      Connect Camera to USB port

10.  Connect Audio jack to USB port

11.  Connect the speaker power USB to the USB hub using a no data USB cable (the short one)

12.  Connect Audio input and output

13.   

14.  cd ~/candy_machine/CandyWebView/node/server

15.  node candy.js &

·        you will be provided with port name add it to URL in browser (e.g. http://192.168.1.17:8080/)
